# AnimeZid CloudStream Plugin

CloudStream extension for [animezid.cam](https://animezid.cam) - Arabic anime and cartoon streaming site.

## Setup Instructions

### 1. Fork & Clone

1. Fork this repo on GitHub (or create a new repo and push these files)
2. Go to `Settings > Actions > General` and enable:
   - "Allow all actions and reusable workflows"
   - "Read and write permissions" (Workflow permissions)

### 2. Update Build Config

In `build.gradle.kts`, replace `3alaouimoha-moh/AnimeZidPlugin` with your GitHub username/repo:

```kotlin
cloudstream {
    setRepo(System.getenv("GITHUB_REPOSITORY") ?: "3alaouimoha-moh/AnimeZidPlugin")
}
```

### 3. Push to GitHub

```bash
git add .
git commit -m "Initial commit"
git push origin master
```

GitHub Actions will build the plugin and deploy it to the `builds` branch.

### 4. Get the Repository URL

After the build completes, your repository.json will be at:
```
https://raw.githubusercontent.com/3alaouimoha-moh/AnimeZidPlugin/master/repository.json
```

Update `repository.json` and `plugins.json` with your actual GitHub username.

### 5. Add to CloudStream

1. Open CloudStream
2. Go to Settings > Extensions
3. Tap "+" icon
4. Enter the repository URL from step 4
5. Tap "Add"
6. Install the AnimeZid plugin

## Files

| File | Description |
|------|-------------|
| `AnimeZidProvider/src/main/kotlin/.../AnimeZidProvider.kt` | The scraper/provider code |
| `repository.json` | Repository definition (add this URL to CloudStream) |
| `plugins.json` | Plugin list (auto-generated by CI on builds branch) |
| `.github/workflows/build.yml` | GitHub Actions build workflow |

## Development

To build locally:

```bash
./gradlew AnimeZidProvider:make
```

To deploy to a connected device:

```bash
./gradlew AnimeZidProvider:deployWithAdb
```
